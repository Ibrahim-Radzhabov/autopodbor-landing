## 🔴 КРИТИЧНО — без этого сайт не работает

### 1. Заменить телефонный номер
**Файлы:** `index.html`, `privacy.html`

Текущий номер-заглушка: `+79001234567` / `+7 900 123-45-67`

В `index.html` найти и заменить:
- `CONFIG.PHONE: '+79001234567'` — в JS CONFIG блоке
- `href="tel:+79001234567"` — ссылки на телефон в шапке, footer, floating widget, sticky CTA
- `https://wa.me/79001234567` — ссылки на WhatsApp
- Текстовое отображение номера в шапке

В `privacy.html`:
- `<a href="tel:+79001234567">+7 900 123-45-67</a>` (строка 106)

### 2. Подключить Telegram Bot API для заявок
**Файл:** `index.html`, JS CONFIG (~строка 2348)

Шаги:
1. Создать бота через @BotFather в Telegram → получить `BOT_TOKEN`
2. Написать боту `/start`, затем открыть `https://api.telegram.org/bot{TOKEN}/getUpdates` — найти `chat.id`
3. Заменить в CONFIG:
   ```javascript
   TELEGRAM_BOT_TOKEN: 'YOUR_BOT_TOKEN',   // → реальный токен
   TELEGRAM_CHAT_ID: 'YOUR_CHAT_ID',       // → реальный chat_id
   ```

### 3. Подключить Web3Forms для email-доставки заявок
**Файл:** `index.html`, JS CONFIG (~строка 2349)

Шаги:
1. Зарегистрироваться на [web3forms.com](https://web3forms.com) (бесплатно, до 250 заявок/мес)
2. Создать форму, получить `access_key`
3. Заменить в CONFIG:
   ```javascript
   WEB3FORMS_KEY: 'YOUR_WEB3FORMS_ACCESS_KEY',  // → реальный ключ
   ```

### 4. Подключить Яндекс.Метрику
**Файл:** `index.html`

Шаги:
1. Создать счётчик на [metrika.yandex.ru](https://metrika.yandex.ru) с включёнными Вебвизором и картой кликов
2. Получить номер счётчика (8-значное число)
3. Заменить в двух местах:
   - В теге `<script>` в `<head>` (~строка 69): `ym(XXXXXXXX, "init", ...)` → `ym(12345678, "init", ...)`
   - В теге `<noscript>` (~строка 76): `watch/XXXXXXXX` → `watch/12345678`
   - В JS CONFIG (~строка 2354): `METRIKA_ID: 'XXXXXXXX'` → `METRIKA_ID: '12345678'`

## 🔵 ДЕПЛОЙ — технические шаги

### 13. Подготовить хостинг
Лендинг — статический сайт (только HTML-файлы). Подходит любой статический хостинг:
- **Простой вариант:** Timeweb, REG.RU, Beget (русские хостинги)
- **Бесплатный вариант:** Netlify, Vercel, GitHub Pages
- **Требования:** поддержка HTTPS (обязательно для работы Метрики и форм)

### 14. Залить файлы
Загрузить на хостинг:
- `index.html`
- `privacy.html`
- `robots.txt`
- `sitemap.xml`
- `og-image.jpg` (если создан)

**НЕ загружать:** `agents.md`, `TODO.md`, `prompt-autopodor-landing-v2.md`, `.claude/`

### 15. Проверить после деплоя
После публикации проверить:
- [ ] Открывается HTTPS (не HTTP)
- [ ] Форма отправляет заявки (проверить в Telegram и email)
- [ ] Яндекс.Метрика фиксирует посещения (раздел «Сводка»)
- [ ] Лендинг корректно выглядит на мобиле (iPhone/Android)
- [ ] OG-карточка отображается при шере ссылки (проверить через vk.com/dev/pages → Share)
- [ ] Нет console.error в браузере

### 16. Зарегистрировать сайт в Яндекс.Вебмастере
После деплоя:
1. Добавить сайт в [webmaster.yandex.ru](https://webmaster.yandex.ru)
2. Подтвердить права (HTML-мета тег)
3. Отправить sitemap.xml
